# EyeMVNet: Uncertainty-Informed Multi-View Network for Image Classification
<p align="justify">
EyeMVNet: Uncertainty-Informed Multi-View Network for Retinal Image Classification


### Objective
<p align="justify">
This project focuses on multi-view retinal image classification and can be readily extended to other general multi-view classification tasks.
</p>


### Methods Used
* Deep Learning
* Machine Learning
* Hierarchical Contrastive Learning

### Technologies
* Python
* Jupyter
* Numpy
* OpenCV
* Pytorch

### Needs of this project
* Deep Learning Modeling
* Evaluation and Reporting


### Getting Started
1. Prepare the Pytorch >=2.0 version and Python >=3.6. Download and install this repo.
```
git clone 
```
2. Move to this repo and install the environment.
```
conda env create -f environment.yml
```
3. Organizing the dataset. The dataset should be organized into the following format. We do not provide the dataset for privacy reasons.
```
Data_directory/'view_name'/
├── class_1/
│   ├── imgid1_name.png
│   ├── imgid2_name.png
│   ├── ...
├── class_2/
│   ├── imgid3_name.png
│   ├── imgid4_name.png
│   ├── ...
├── class_3/
│   ├── imgid5_name.png
│   ├── imgid6_name.png
│   ├── ...
└── ...

```
4. Launch the Jupyter Notebook and change the param according to your settings.
